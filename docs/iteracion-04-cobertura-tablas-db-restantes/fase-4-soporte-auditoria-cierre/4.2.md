# 4.2 - Implementar auditoria minima (audit_logs)

## Objetivo

Implementar registro de eventos de auditoria minimos y endpoint de consulta del propio usuario.

## Razon Tecnica

Mejora trazabilidad funcional y completa cobertura de tabla audit_logs sin sobrediseno.

## Comandos Exactos

```bash
rg -n "CREATE TABLE public\.audit_logs|CREATE POLICY .*audit_logs" humanis_db_init.sql
./mvnw -DskipTests=false -Dtest=*Audit*Test test
```

## Ejemplo De Codigo (Si Aplica)

## Verificacion

1. Se crean registros de auditoria en acciones clave.
2. Consulta de logs respeta ownership por usuario.
3. Pruebas de auditoria en verde.

Auditoria minima funcional sin impactar arquitectura modular.
