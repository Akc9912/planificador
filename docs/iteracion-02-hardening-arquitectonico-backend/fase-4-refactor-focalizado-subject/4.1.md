# 4.1 - Extraer reglas de correlativas de SubjectService

## Objetivo

Reducir complejidad de SubjectService moviendo reglas de correlativas a un componente dedicado de dominio/politica.

## Razon Tecnica

SubjectService concentra demasiada logica y aumenta riesgo de regresiones al modificar reglas academicas.

## Comandos Exactos

```bash
rg -n "validateCorrelativeRulesForStatus|isBlocked|isCorrelativeApproved|toCorrelativeList" src/main/java/aktech/planificador/modules/subject/application/SubjectService.java
cat src/main/java/aktech/planificador/modules/subject/application/SubjectService.java
./mvnw -DskipTests=false -Dtest=SubjectServiceTest test
```

## Verificacion

1. SubjectService reduce metodos internos de regla de correlativas.
2. Reglas extraidas mantienen comportamiento funcional.
3. SubjectServiceTest permanece en verde.

## Resultado Esperado

Servicio de aplicacion mas pequeno y reglas de correlativas encapsuladas en componente de dominio.
