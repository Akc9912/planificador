# 2.3 - Ejecutar verificacion de arquitectura y registrar evidencia

## Objetivo

Ejecutar validacion completa de guardrails y dejar evidencia consolidada en documentacion de la iteracion.

## Razon Tecnica

Toda regla nueva debe cerrarse con evidencia reproducible para evitar dudas sobre su efectividad.

## Comandos Exactos

```bash
./mvnw -DskipTests compile
./mvnw -DskipTests=false -Dtest=ModuleBoundariesTest test
./mvnw -DskipTests=false -Dtest=PlanificadorApplicationTests test
```
