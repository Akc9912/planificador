# 2.1 - Agregar dependencias y config base de Redis

## Objetivo

Incorporar Redis al backend con configuracion minima necesaria para soportar rate limiting distribuido.

## Razon Tecnica

Rate limit en memoria local no escala ni se comporta bien en multiples instancias; Redis permite contador compartido.

## Comandos Exactos

```bash
cat pom.xml
rg -n "spring-boot-starter-data-redis|lettuce|jedis" pom.xml
cat src/main/resources/application.properties
./mvnw -DskipTests compile
```

## Ejemplo De Codigo (Si Aplica)

```txt
Dependencia sugerida:
- org.springframework.boot:spring-boot-starter-data-redis

Propiedades sugeridas:
- spring.data.redis.host=${REDIS_HOST}
- spring.data.redis.port=${REDIS_PORT}
```

## Verificacion

1. Dependencia Redis agregada en pom.xml.
2. Configuracion Redis declarada en application.properties.
3. Compilacion en verde tras integrar dependencia.

## Resultado Esperado

Proyecto preparado para conectar Redis desde infraestructura.
