# 1.1 - Medir baseline de trafico y endpoints sensibles

## Objetivo

Establecer baseline tecnico para justificar limites y evitar configuraciones de rate limit arbitrarias.

## Razon Tecnica

La introduccion de seguridad por limite requiere metricas base para no bloquear uso legitimo ni dejar huecos de abuso.

## Comandos Exactos

```bash
./mvnw -DskipTests compile
./mvnw spring-boot:run
# en otra terminal, ejecutar rafagas de prueba sobre endpoints publicos
for i in {1..30}; do curl -s -o /dev/null -w "%{http_code}\n" http://localhost:8081/v3/api-docs; done
for i in {1..30}; do curl -s -o /dev/null -w "%{http_code}\n" -X POST http://localhost:8081/auth/token/validate -H "Content-Type: application/json" -d '{"accessToken":"invalid"}'; done
```
